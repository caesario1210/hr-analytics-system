# Dashboard Guide — HR Analytics System

Build an interactive spreadsheet dashboard from analysis results.

## Step 1: Export Analysis Results

```powershell
cd "D:\RIO\PORTOFOLIO\HR Analyst System"
python dataset/export_analysis.py
```

Output: `dashboard/analysis_results/*.csv` (16 files across 5 categories)

## Step 2: Create Spreadsheet

Open `dashboard/HR Dashboard.xlsx` (new file) and create 5 sheets:

### Sheet 1: Employee Overview

| Data Source | Chart Type | Fields |
|-------------|-----------|--------|
| `analysis_results/emp_status.csv` | Donut | status, total |
| `analysis_results/emp_department.csv` | Bar | department_name, total_employees |
| `analysis_results/emp_gender.csv` | Pie | gender, percentage |
| `analysis_results/emp_active_dept.csv` | Bar | department_name, active_employees |

KPI cards: Total =SUM(), Active count, % Active.

### Sheet 2: Payroll

| Data Source | Chart Type | Fields |
|-------------|-----------|--------|
| `analysis_results/payroll_overall.csv` | KPI card | mean, min, max |
| `analysis_results/payroll_department.csv` | Bar | department_name, mean |
| `analysis_results/payroll_top10.csv` | Horizontal bar | full_name, salary |
| `analysis_results/payroll_rank.csv` | Table | full_name, department_name, salary, salary_rank |

### Sheet 3: Attendance

| Data Source | Chart Type | Fields |
|-------------|-----------|--------|
| `analysis_results/attendance_status.csv` | Donut | status, percentage |
| `analysis_results/attendance_monthly.csv` | Line | month (x), total (y), status (series) |
| `analysis_results/attendance_late.csv` | Bar | full_name, late_days |

### Sheet 4: Recruitment

| Data Source | Chart Type | Fields |
|-------------|-----------|--------|
| `analysis_results/recruitment_funnel.csv` | Funnel bar | status, candidates |
| `analysis_results/recruitment_source.csv` | Bar | source, hire_rate_percent |
| `analysis_results/recruitment_timetohire.csv` | KPI card | avg_days_to_hire |

### Sheet 5: Performance

| Data Source | Chart Type | Fields |
|-------------|-----------|--------|
| `analysis_results/performance_period.csv` | Line | period (x), kpi_score + manager_score (series) |
| `analysis_results/performance_top10.csv` | Bar | full_name, avg_kpi |
| `analysis_results/performance_bottom10.csv` | Bar (reverse) | full_name, avg_kpi |
| `analysis_results/training_course_avg.csv` | Bar | course_name, avg_score |

## Step 3: Layout Tips

- Header row: dark blue fill (#1e3a5f), white bold text
- KPI cards: large font, border, conditional formatting (green/red arrows)
- Consistent color palette per sheet
- Add slicers/filters for interactivity (filter by department, period)
